Credenciais de login
Username -> login
Password -> admin
No script fornecido não é criardo um utilizador, visto que as passwords são encriptadas na criação de um user. Logo, para iniciar o script é necessário criar as seguintes contas, com os seguintes usernames e passwords:
	admin    admin
	diogo    1234
	miguel   1234
	ze       1234
Todos os outros elementos podem ser aleatórios e podem ser alterados na página dee perfil de um usuário.
	

Apenas são utilizados ícones presentes em http://fontawesome.io/. São utilizados para auxiliar na fácil compreensão de alguns inputs e como botões.
